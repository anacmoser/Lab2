# Logic3 - Programming Laboratory 2
This repository contains the materials, exercises, and projects from the **Programming Laboratory 2** subject, taught by Prof. Domingos Bernardo, for students at IFSP.

---

## About the Subject

In this subject, we explore **Object-Oriented Programming (OOP)** applied to real software development. The goal is to understand and apply OOP concepts in software projects using layered architecture, design patterns, and data persistence.

The course is hands-on, with a practical incremental project developed throughout the semester. Topics include building structured and object-oriented software, developing web applications, applying design patterns such as **MVC**, working with **database persistence**, and writing professional, organized code.

Through this subject, we develop skills to build complete, well-structured systems — from the data layer to the web interface.

---

## Modules

- **Module 1 – Language Fundamentals:** Variables, constants, data types, operators, control structures, collections (List, Set, Map), exception handling, and file I/O.
- **Module 2 – Object-Oriented Programming:** Classes, objects, encapsulation, inheritance, polymorphism, and interfaces.
- **Module 3 – Web Programming:** Web application architecture, client-server model, back-end languages, frameworks, web servers, and deploy.
- **Module 4 – Design Patterns:** MVC, ORM, data persistence, coupling, and separation of responsibilities.
